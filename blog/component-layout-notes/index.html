Component layout notes and accessibility pass

Published February 6, 2026

Updated February 9, 2026

  • #design
  • #css
  • #accessibility

This entry records decisions made while shaping the current layout.

Layout rhythm

  • Content width is constrained to improve reading flow.
  • Cards use consistent radius, border, and shadow tokens.
  • Header and footer are simple to reduce chrome noise.

Accessibility notes

  • Structural landmarks are explicit: header, main, and footer.
  • Navigation has aria-label="Primary".
  • Link colors are tested against both light and dark backgrounds.

Responsive behavior

At small widths, navigation wraps and the header switches to a vertical stack.

@media (max-width: 640px) {
  .header-inner {
    flex-direction: column;
    align-items: flex-start;
  }
}
